Extra Scheme Components Plugin
Version       : 5.1.1.0
Home page     : https://rapidscada.org/download-all-files/download-extra-scheme-components-plugin/


Installation
------------
1. Perform the general sequence of installing plugins 
https://rapidscada.net/doc/content/latest/en/installation-and-run/module-installation.html#plugins
2. Go to Main Menu > Registration > Extra Scheme Components.
3. Register the plugin using the hyperlinks on the page.
